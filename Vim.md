# Vim Cheat sheet

## Copy-paste

| Key          | Action                   |
| :----------- | :----------------------- |
| `"*y` `"*p`  | Copy/paste OS clipboard  |

## Split screen

| Key           | Action                   |
| :-----------  | :----------------------- |
| `C-w s`       | horizontal split         |
| `C-w v`       | vertical split           |
| `C-w q`       | close split              |
| `C-w up/down` | switch split             |
| `C-w C-w`     | cycle splits             |

## Leader shortcuts

(Leader = ,)

| Key    | Action    |
| :----- | :-------- |
| `,t`   | CommandT  |
| `,ww`  | Wikii     |
